网站与网页

网站：互联网上用于展示特定内容的相关网页集合。

网页：网站中的一页，一个网站中的网页通过“超链接”的方式被组织在一起。

主页的文件名通常是index

页面元素：LOGO站标、导航栏、文字链接、banner广告横幅、表单

从开发者角度：

网站：文件夹 网页：文件

浏览器：解析网页源代码、渲染网页。读取 HTML 文档，并以网页的形式显示出它们。
